# Kaushik Somashekar Portfolio

GitHub Pages-ready personal portfolio.

## Upload steps
1. Extract this ZIP.
2. Upload `index.html`, `styles.css`, `script.js`, and the `assets` folder to your GitHub Pages repository.
3. Commit changes.
4. Open `https://kaushiks1.github.io/`.
